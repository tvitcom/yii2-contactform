-------(Ubuntu all???)Preferable variant[many version in one machine]:---------
sudo apt-get update
sudo apt-get install build-essential libssl-dev
curl https://raw.githubusercontent.com/creationix/nvm/v0.16.1/install.sh | sh
И каждый раз перед применением в терминале нужно набрать запуск переменных среды:
source ~/.nvm/nvm.sh
И использовать:
cd /srv/Sites/expres/webroot
nvm install v0.10.24        //Install a specific version number
nvm use 0.10  
nvm use 0.11.13              //Use the latest available 0.10.x release
nvm run 0.10.24 myApp.js    //Run myApp.js using node v0.10.24
-------(Ubuntu 14) Install nodejs 4.x:--------------

node --version
